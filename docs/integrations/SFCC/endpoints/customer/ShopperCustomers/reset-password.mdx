# Reset Password

## Purpose

This endpoint is used to reset a customer's password.

:::note

Resetting a customer's password depends on the token from the [ShopperLogin.getPasswordResetToken](../ShopperLogin/get-password-reset-token.mdx) endpoint.

:::

## Example

<Tabs>
  <TabItem value="commerce-sdk" label="Commerce SDK">

## Request

### URL

`POST https://{shortCode}.api.commercecloud.salesforce.com/customer/shopper-customers/{version}/organizations/{organizationId}/customers/password/actions/reset?siteId={siteId}`

### Headers

- `Authorization: Bearer {slas_token}`
- `Content-Type: application/json`

### URL Parameters

- `shortCode` _string_: A region-specific merchant identifier. **(required)**
  - Retrieved from Business Manager > Administration > Salesforce Commerce API Settings > Shortcode
  - Example: `0dnz6oep`
- `version` _string_: API version identifier. **(required)**
  - Example: `v1`
- `organizationId` _string_: The identifier for the organization. **(required)**
  - Retrieved from Business Manager > Administration > Salesforce Commerce API Settings > organization ID

### Query Parameters

- `siteId` _string_: Identifies the site context of the request. Attributes may have site-specific values. **(required)**
  - Example: `RefArch`

### Request Body

```json
{
  "resetToken": "R1e2s3e4t5T6o7k8e9n0",
  "login": "janedoe@test.com",
  "newPassword": "p@assword2"
}
```
